aggregateByRegions

  • aggregateByRegions(result: Result, model: Model): Promise<[Sector[], Result]>

  • If the given result is a multi-regional model, this function aggregates the given result so that the result columns of the same sector in different regions are summed up to a single column. Together with the aggregated result a corresponding sector array is returned but this sector information should be never used for API requests as they describe an artificialy aggregated sector. If the given model describes a single region, the result and sectors are returned without modification.

    Parameters

    • result: Result
    • model: Model

    Returns Promise<[Sector[], Result]>
