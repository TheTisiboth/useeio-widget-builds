Type aliases

Matrix

Matrix: "D" | "U" | "RD" | "RU"

Mnemonics for the different matrices:

D : direct result matrix based on 1 USD for each sector U : matrix with upstream totals based on 1 USD for each sector RD: direct result matrix based on a demand vector RU: matrix with upstream totals based on a demand vector
