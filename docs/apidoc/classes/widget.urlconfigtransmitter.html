Class UrlConfigTransmitter

A ConfigTransmitter implementation that reads and serializes its configuration state from and to the hash part of the current URL.

Private patchHash

  • patchHash(configHash: string): void

  • Update the current hash with the given value but keep current hash attributes that are no configuration attributes in the hash. Note that configuration attributes can be dropped intentionally from the hash part (indicating that something is not set). Also, configuration attributes can have scope prefixes. Thus, it is not enough to just keep the current attributes in the has; we need to check for each current attribute if it is a configuration attribute instead.

    Parameters

    • configHash: string

    Returns void
